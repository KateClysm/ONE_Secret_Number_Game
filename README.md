# 🎯 Juego del Número Secreto

Este proyecto fue desarrollado como parte del programa **Oracle Next Education** en conjunto con **Alura Latam**. El objetivo del juego es adivinar un número aleatorio generado por el sistema. El jugador recibe pistas dinámicas para acercarse a la respuesta correcta, utilizando elementos de programación interactiva con JavaScript.

---

## 🚀 Tecnologías y habilidades aplicadas

Durante el desarrollo del proyecto, puse en práctica los conocimientos adquiridos en los siguientes cursos:

### 📘 Curso: *Lógica de programación: sumérgete en la programación con JavaScript*
- Mostrar mensajes en pantalla de forma interactiva.
- Utilización de variables para desarrollar software.
- Desarrollo de aplicaciones siguiendo buenas prácticas de programación.
- Aplicación construida de principio a fin, inspirada en un escenario del mundo real.
- Adaptación de soluciones proporcionadas por el lenguaje JavaScript.

### 📗 Curso: *Lógica de programación: explorar funciones y listas*
- Uso de funciones para manipular elementos de la interfaz.
- Uso de listas (arrays) como estructura de datos clave.
- Publicación del proyecto usando GitHub.
- Manipulación dinámica de palabras y números con JavaScript.

### 🛠️ Curso: *Git y GitHub: repositorio, commit y versiones*
- Control de versiones del proyecto mediante Git.
- Comandos fundamentales como `git add`, `commit`, `push` y `log`.
- Creación y gestión de ramas.
- Publicación del proyecto en GitHub Pages.
